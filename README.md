# -
Репозиторий посвящен кластеризации клиентов банка по использованию кредитных карт. С помощью K-Means выделены группы: активные покупатели, дисциплинированные плательщики и группа риска (минимальные платежи). Внутри: EDA, очистка данных, масштабирование признаков и профилирование кластеров для маркетинга.
